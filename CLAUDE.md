# aunitz.github.io — instrucciones para Claude Code

## Idioma
Responde siempre en español.

## Git
- Nunca hagas `git commit` ni `git push`. El usuario los hace siempre manualmente.
- No propongas commits a menos que el usuario lo pida explícitamente.

## Proyecto
Blog personal de Aunitz Giménez en Jekyll/GitHub Pages (aunitz.net). Temas: inteligencia artificial, UX y diseño centrado en el usuario.

## Publicación de posts
Para crear o publicar un nuevo post, usa siempre la skill `publish-post`. No lo hagas manualmente.

## Estilo de código
- Sin comentarios salvo que el motivo sea no obvio.
- Sin emojis salvo petición explícita.
- Respuestas cortas y directas.
